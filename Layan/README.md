# Layan - Create Balagh Screen

## المسؤولية
شاشة إنشاء البلاغ (Create Balagh)

## الوصف
هذه الشاشة تتيح للطلاب والموظفين إنشاء بلاغات صيانة جديدة

## المميزات
- ✅ إدخال عنوان البلاغ
- ✅ اختيار قسم الحرم الجامعي (طلاب/طالبات)
- ✅ اختيار فئة المشكلة (كهرباء، أثاث، تكييف، إلخ)
- ✅ اختيار المبنى
- ✅ إدخال تفاصيل الموقع
- ✅ وصف المشكلة
- ✅ إرفاق صورة
- ✅ إرسال البلاغ

## الملفات
- `create_balagh_screen.dart` - الشاشة الرئيسية لإنشاء البلاغ

## التقنيات المستخدمة
- Flutter
- Dart
- RTL Support (دعم اللغة العربية)
- Image Picker (اختيار الصور)
